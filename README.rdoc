= JSRuby

A Ruby 1.9 virtual machine in JavaScript (thanks to node.js).

Influenced by HotRuby: http://hotruby.yukoba.jp/

= Requirements

- node.js (of course)
- Ruby 1.9.2 (YARV), for actually compiling the ruby source code to opcodes.
